ΑΘΑΝΑΣΙΑ ΚΑΒΒΑΔΙΑ/ΙΕΠ 2023

Στο δείγμα 1 περιέχεται ο Ανοικτός Εκπαιδευτικός Πόρος 1 με τίτλο: Η θερμοκρασία των Αθηνών. Ο συγκεκριμένος εκπαιδευτικός πόρος ανήκει στην κατηγορία εφαρμογές λογισμικού.
